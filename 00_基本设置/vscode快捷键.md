# VS Code 最常用的快捷键

## 主题

> **Material Theme Oceanic custom for VS Code**

以下是 Visual Studio Code (VS Code) 中最常用的快捷键，按类别分组整理。这些快捷键适用于 Windows 系统（Mac 和 Linux 可能略有不同）。

## 通用操作

- `Ctrl + Shift + P` / `F1`: 打开命令面板
- `Ctrl + P`: 快速打开文件
- `Ctrl + ,`: 打开用户设置
- `Ctrl + K Ctrl + S`: 打开键盘快捷键设置

## 编辑操作

- `Ctrl + Shift + U`: 转为大写 Ctrl + K Ctrl + S Transform to Uppercase
- `Ctrl + U`: 转为小写 Ctrl + K Ctrl + S Transform to Lowercase
- `Ctrl + Shift + S`: 另存为
- `Ctrl + F`: 在文件中查找
- `Ctrl + H`: 在文件中替换
- `Ctrl + Shift + F`: 在工作区中查找
- `Ctrl + Shift + H`: 在工作区中替换
- `Alt + Shift + F`: 格式化文档
- `Ctrl + /`: 切换行注释
- `Shift + Alt + A`: 切换块注释
- `Ctrl + Shift + K`: 删除行
- `Alt + ↑/↓`: 移动行上下
- `Shift + Alt + ↑/↓`: 复制行上下
- `Ctrl + Enter`: 在下方插入行
- `Ctrl + Shift + Enter`: 在上方插入行
- `Ctrl + D`: 选择下一个匹配项
- `Ctrl + Shift + L`: 选择所有匹配项
- `Ctrl + F2`: 选择所有出现
- `F2`: 重命名符号

## 导航操作

- `Ctrl + G`: 转到行
- `Ctrl + Shift + O`: 转到符号
- `Ctrl + T`: 转到工作区符号
- `Ctrl + Tab`: 切换标签页
- `Ctrl + W`: 关闭标签页
- `Ctrl + Shift + T`: 重新打开关闭的标签页
- `Ctrl + K Ctrl + O`: 打开文件夹
- `Ctrl + K Ctrl + C`: 关闭文件夹

## 视图操作

- `Ctrl + B`: 切换侧边栏
- `Ctrl + J`: 切换面板
- `Ctrl + Shift + M`: 切换问题面板
- `Ctrl + Shift + Y`: 切换调试控制台
- `Ctrl + ``: 切换集成终端
- `Ctrl + Shift + ``: 新建终端

## 调试操作

- `F5`: 开始调试 / 继续调试
- `Ctrl + F5`: 运行（不调试）
- `Shift + F5`: 停止调试
- `F9`: 切换断点
- `F10`: 单步执行（逐过程）
- `F11`: 单步执行（逐语句）
- `Shift + F11`: 单步跳出

## 多光标和选择

- `Alt + Click`: 插入光标
- `Ctrl + Alt + ↑/↓`: 插入光标上下
- `Ctrl + Shift + Alt + ↑/↓`: 列选择上下
- `Shift + Alt + (drag mouse)`: 列选择

## 其他常用

- `Ctrl + K Ctrl + F`: 格式化选定代码
- `Ctrl + K Ctrl + C`: 添加行注释
- `Ctrl + K Ctrl + U`: 删除行注释
- `Ctrl + Shift + P`: 再次强调命令面板
- `Ctrl + Shift + I`: 显示悬停信息

这些快捷键可以大大提高编码效率。建议根据个人习惯自定义快捷键设置。
